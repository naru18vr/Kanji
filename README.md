# まいにち漢字トレーニング

子どもが毎日迷わず続けられる、漢検10級〜3級向けの静的PWAです。学習記録は端末内だけに保存し、Google Chatへの送信は利用者がコピー内容を確認して行います。

ホームから「漢検級別」と「学年別」のどちらでもコースを選べます。全画面に共通の「もどる」「ホーム」操作があり、Androidの戻る操作でも学習途中の回答を保持します。

## 開発

```bash
npm ci
npm test
npm run build
```

GitHub Pages: https://naru18vr.github.io/Kanji/

## データとプライバシー

- 保存キー: `kanji-study-v1`
- 現行スキーマ: version 1
- 不正なデータを無視し、ID単位で重複を排除します。
- 容量不足時は sessionStorage に一時退避します。
- 氏名・学校名・住所・APIキー・外部トラッキングを使用しません。
- 引き継ぎリンクには学習記録が含まれるため、公開場所へ貼らないでください。
